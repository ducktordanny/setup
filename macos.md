# MacOS Setup

> My MacOS setup for web and mobile developement

## Installations

### HomeBrew

```sh
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

### Google Chrome

```sh
brew install --cask google-chrome
```

### Git

```sh
brew install git
```

### Node, npm and yarn

Node and npm

```sh
brew install node
```

Yarn

```sh
npm install --global yarn
```

## Visual Studio Code
